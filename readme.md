# [imluke.me](http://imluke.me/)

## Description

tumblr theme and assets for [my personal blog](http://imluke.me/). `tumblr.html` contains the actual code blocks used for tumblr; `index.html` contains the full HTML â€” i.e. the way the theme should render on tumblr with actual post information. `tumblr.html` loads assets from the `gh-pages` branch on GitHub (so that I can easily push/pull/manage assets); `index.html` loads the local files.

## Installation/Build

```bash
## install dependencies
npm install
## compile js/less
grunt default
```
